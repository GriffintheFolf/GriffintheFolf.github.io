Tech Projects

Because why not

Technology has long fascinated me, and I figure it's time I help out the community.

Much of the interest here revolves around old technology, such as PowerPC Macs, although it also includes more modern systems such as Haiku.

Most of this relies on porting existing software. While I have written some programs of my own, they certainly do not compare to the existing solutions.

PowerPC Ports

This section documents some software I have ported to PowerPC Macs running Mac OS X.

Unless otherwise specified, the software requires the following:

  • PowerPC Mac
  • PowerPC G3, G4, or G5
  • Mac OS X 10.4.11 Tiger

Some software may have specific builds optimized for a particular processor; you should download the one appropriate for yours.
If no specific builds are listed, then the software will work on any PowerPC processor of at least a G3.

These ports should also work on Mac OS X 10.5.8 Leopard; note that running Leopard implies the usage of at least a G4.

Lastly, these ports may run on an Intel-based Mac running 10.4.4 Tiger to 10.6.8 Snow Leopard through Rosetta.
However, this has not been tested and may lead to suboptimal performance.
